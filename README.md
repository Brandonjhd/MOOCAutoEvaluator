## 🇨🇳 简体中文

自动完成中国大学MOOC作业互评 —— 提升学习效率的智能助手

### 简介

MOOCAutoEvaluator 是一个用于自动完成中国大学MOOC平台作业互评的用户脚本工具。通过智能化的自动评价流程，帮助学生高效完成互评任务，节省时间专注于学习本身。

### 主要功能

- ✅ **一键自动评价** - 自动选择分数、填写评语、提交作业
- 🤖 **AI提示词生成** - 根据课程和作业名称自动生成AI评价提示词
- 📊 **智能进度管理** - 实时显示剩余评价数量和已完成进度
- 🔄 **循环评价机制** - 评价完成后自动删除已使用的评语
- ⏸️ **随时暂停控制** - 支持一键停止，操作灵活可控
- 💬 **批量评语支持** - 支持导入多条评语循环使用
- 🎯 **边界智能检测** - 评语用完或作业完成后自动停止

### 安装使用

#### 1. 安装浏览器扩展
首先需要安装用户脚本管理器（任选其一）：
- [Tampermonkey](https://www.tampermonkey.net/) (推荐)
- [Violentmonkey](https://violentmonkey.github.io/)
- [Greasemonkey](https://www.greasespot.net/)

#### 2. 安装脚本
点击下方链接安装脚本：
- [从 GitHub 安装](https://github.com/Brandonjhd/MOOCAutoEvaluator/raw/main/mooc-auto-evaluator.user.js)

#### 3. 使用步骤
1. 打开中国大学MOOC平台的作业互评页面
2. 点击页面右侧的紫色悬浮球 ✓
3. 点击「复制AI提示词」按钮，粘贴到ChatGPT等AI工具生成评语
4. 将生成的评语粘贴到文本框中（每行一条）
5. 点击「开始评价」按钮，脚本会自动完成所有互评

## 🇺🇸 English

Automated peer review tool for China University MOOC platform — Boost your learning efficiency

### Introduction

MOOCAutoEvaluator is a userscript tool designed to automate peer review assignments on the China University MOOC platform. It streamlines the evaluation process, helping students complete peer reviews efficiently and focus more on learning.

### Features

- ✅ **One-Click Auto Evaluation** - Automatically selects scores, fills in comments, and submits assignments
- 🤖 **AI Prompt Generator** - Auto-generates AI evaluation prompts based on course and assignment names
- 📊 **Smart Progress Management** - Real-time display of remaining evaluations and completion progress
- 🔄 **Cyclic Evaluation** - Automatically removes used comments after evaluation
- ⏸️ **Pause Control** - Supports one-click stop for flexible operation
- 💬 **Batch Comments Support** - Import multiple comments for cyclic use
- 🎯 **Smart Boundary Detection** - Auto-stops when comments run out or all assignments are completed

### Installation

#### 1. Install Browser Extension
First, install a userscript manager (choose one):
- [Tampermonkey](https://www.tampermonkey.net/) (Recommended)
- [Violentmonkey](https://violentmonkey.github.io/)
- [Greasemonkey](https://www.greasespot.net/)

#### 2. Install Script
Click the link below to install:
- [Install from GitHub](https://github.com/Brandonjhd/MOOCAutoEvaluator/raw/main/mooc-auto-evaluator.user.js)

#### 3. Usage
1. Open the peer review page on China University MOOC platform
2. Click the purple floating ball ✓ on the right side
3. Click "Copy AI Prompt" and paste it into ChatGPT or other AI tools
4. Paste the generated comments into the text box (one per line)
5. Click "Start Evaluation" and the script will complete all reviews automatically

## 🇯🇵 日本語

中国大学MOOCプラットフォームの相互評価を自動化 — 学習効率を向上させるスマートアシスタント

### 概要

MOOCAutoEvaluator は、中国大学MOOCプラットフォームの課題相互評価を自動化するユーザースクリプトツールです。スマートな自動評価プロセスにより、学生が効率的に相互評価タスクを完了し、学習そのものに集中できるよう支援します。

### 主な機能

- ✅ **ワンクリック自動評価** - 自動でスコア選択、コメント入力、課題提出
- 🤖 **AIプロンプト生成** - コースと課題名に基づいてAI評価プロンプトを自動生成
- 📊 **スマート進捗管理** - 残りの評価数と完了進捗をリアルタイム表示
- 🔄 **循環評価メカニズム** - 評価完了後、使用済みコメントを自動削除
- ⏸️ **いつでも一時停止** - ワンクリック停止機能で柔軟な操作
- 💬 **バッチコメント対応** - 複数のコメントをインポートして循環使用
- 🎯 **スマート境界検出** - コメントがなくなるか全ての課題が完了すると自動停止

### インストール

#### 1. ブラウザ拡張機能をインストール
まず、ユーザースクリプトマネージャーをインストールします（いずれか選択）：
- [Tampermonkey](https://www.tampermonkey.net/) (推奨)
- [Violentmonkey](https://violentmonkey.github.io/)

#### 2. スクリプトをインストール
以下のリンクからインストール：
- [GitHubからインストール](https://github.com/Brandonjhd/MOOCAutoEvaluator/raw/main/mooc-auto-evaluator.user.js)

#### 3. 使用方法
1. 中国大学MOOCプラットフォームの相互評価ページを開く
2. 右側の紫色のフローティングボール ✓ をクリック
3. 「AIプロンプトをコピー」ボタンをクリックし、ChatGPTなどのAIツールに貼り付け
4. 生成されたコメントをテキストボックスに貼り付け（1行に1つ）
5. 「評価開始」ボタンをクリックすると、スクリプトが自動的にすべての評価を完了

중국 대학 MOOC 플랫폼 과제 상호평가 자동화 — 학습 효율을 높이는 스마트 도우미

### 소개

MOOCAutoEvaluator는 중국 대학 MOOC 플랫폼의 과제 상호평가를 자동화하는 사용자 스크립트 도구입니다. 지능형 자동 평가 프로세스를 통해 학생들이 효율적으로 상호평가 작업을 완료하고 학습 자체에 집중할 수 있도록 지원합니다.

### 주요 기능

- ✅ **원클릭 자동 평가** - 자동으로 점수 선택, 코멘트 작성, 과제 제출
- 🤖 **AI 프롬프트 생성** - 과정 및 과제명을 기반으로 AI 평가 프롬프트 자동 생성
- 📊 **스마트 진행 관리** - 남은 평가 수와 완료 진행률을 실시간 표시
- 🔄 **순환 평가 메커니즘** - 평가 완료 후 사용된 코멘트 자동 삭제
- ⏸️ **언제든지 일시정지** - 원클릭 정지 기능으로 유연한 조작
- 💬 **배치 코멘트 지원** - 여러 코멘트를 가져와 순환 사용
- 🎯 **스마트 경계 감지** - 코멘트가 소진되거나 모든 과제가 완료되면 자동 정지

### 설치

#### 1. 브라우저 확장 프로그램 설치
먼저 사용자 스크립트 관리자를 설치합니다(하나 선택):
- [Tampermonkey](https://www.tampermonkey.net/) (권장)
- [Violentmonkey](https://violentmonkey.github.io/)

#### 2. 스크립트 설치
아래 링크에서 설치:
- [GitHub에서 설치](https://github.com/Brandonjhd/MOOCAutoEvaluator/raw/main/mooc-auto-evaluator.user.js)

#### 3. 사용 방법
1. 중국 대학 MOOC 플랫폼의 상호평가 페이지 열기
2. 오른쪽의 보라색 플로팅 버튼 ✓ 클릭
3. "AI 프롬프트 복사" 버튼을 클릭하고 ChatGPT 등 AI 도구에 붙여넣기
